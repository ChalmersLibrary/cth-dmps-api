Search API for Chalmers DSW DMPs. Authorization key required.
Only GET method is supported. Returns RDA Commons maDMP v 1.0 JSON. The following endpoints are valid:

  • /search?q= - Search query in standard Lucence syntax, i.e. "title:my data plan"
  • /plans/ - Retrieve specific plan by ID, i.e. "/3dd145-dgashgd6-476yegh67-sjs6jhd"
